## 单例模式（Singleton Pattern）

* 饿汉模式

  私有构造方法，类加载后初始化示例，缺点消耗更多的内存

  ~~~java
  public class EagerSingleton{
      private static final EagerSingleton instance = new EagerSingleton();
      private EagerSingleton(){
          
      }
      public static EagerSingleton getIntance(){
          return instance;
      }
  }
  ~~~

  

* 懒汉模式

  使用时初始化，延迟加载；线程不安全

  ~~~java
  public class LazySingleton{
      private static final LazySingleton instance;
      private LazySingleton(){
          
      }
      public static LazySingleton getIntance(){
          if(instance == null){
              instance = new LazySingleton();
          }
          return instance;
      }
  }
  ~~~

  

* 懒汉模式双检锁

  ~~~java
  public class DCLSingleton{
      private static final DCLSingleton instance;
      private DCLSingleton(){
          
      }
      public static DCLSingleton getIntance(){
          if(instance == null){
              synchronize(DCLSingleton.class){
                  if (instance == null) {
                      instance = new LazySingleton();
                  }
              }
          }
          return instance;
      }
  }
  ~~~

  

* 懒汉模式-静态内部类

  懒加载，线程安全

  ~~~java
  public class HolderSingleton{
      private HolderSingleton(){
          
      }
      private static class Holder{
          private static final HolderSingleton instance = new HolderSingleton() ;
      }
      public LazySingleton getIntance(){
          return Holder.instance;
      }
  }
  ~~~

  

* 懒汉模式-枚举

  ~~~java
  public enum EnumSingleton {
      INSTANCE;
      public void doSomething() { 
      }
  }
  ~~~

* 性能与安全对比

    |   **实现方式**   | **线程安全** | **延迟加载** | **性能** | **防反射/序列化** |
    | :--------------: | :----------: | :----------: | :------: | :---------------: |
    |      饿汉式      |      ✅       |      ❌       |    高    |         ❌         |
    | 懒汉式（基础版） |      ❌       |      ✅       |    中    |         ❌         |
    |    双重检查锁    |      ✅       |      ✅       |    高    |         ❌         |
    |    静态内部类    |      ✅       |      ✅       |    高    |         ❌         |
    |       枚举       |      ✅       |      ❌       |    高    |         ✅         |
