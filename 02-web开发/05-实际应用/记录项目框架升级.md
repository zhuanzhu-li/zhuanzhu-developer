## 1、Java 8 升级到 Java 17

* **全局检查pom配置**

  关键字：`java.version`、`maven.compiler.source`、`maven.compiler.target`

* **IDEA配置项目为 JDK17 版本**

* **jcoco版本升级**

* **HBase兼容**

  jvm启动参数增加：

  ```
  --add-exports=java.security.jgss/sun.security.krb5=ALL-UNNAMED
  ```
  在项目所在盘下增加 `/etc/krb5.conf.d` 文件夹
  
  当前这里还不能正常连接hbase
  
* sun 包替换、Base64替换
